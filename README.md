## 🧹+🚀 一键切换到哪吒探针

在 VPS 上执行以下命令，即可卸载 Beszel 并安装哪吒 Agent：

```bash
curl -O https://raw.githubusercontent.com/taviswoo/switch-to-nezha.sh/main/switch-to-nezha.sh
chmod +x switch-to-nezha.sh
./switch-to-nezha.sh
```

## 哪吒探针安全检测

```bash
wget -O nezha-check.sh https://raw.githubusercontent.com/taviswoo/switch-to-nezha.sh/refs/heads/main/nezha-check.sh
chmod +x nezha-check.sh
./nezha-check.sh
```
